# goarping

arping equivalent but in Go !, currently supports IPv4 only though ... its just a learning project anyways use arping lol.

```
sudo ./goarping 

Usage of ./goarping:
  -i string
        network interface
  -ip string
        target ip
  -t duration
        timeout (default 2s)

example:
    sudo ./goarping -i=wlo1 -ip=192.168.1.5
```
